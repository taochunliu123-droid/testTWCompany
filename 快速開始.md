## 🎯 兩種 API 使用建議

### 何時使用「第三方 API」？
- ✅ 想要快速測試
- ✅ 不想申請權限
- ✅ 查詢常見大公司

### 何時使用「財政部資料」？
- ✅ 需要官方權威資料
- ✅ 查詢較小或較新的公司
- ✅ 已申請到 API 權限

## 💡 常見問題

### Q: 為什麼查不到資料？

**A**: 可能原因：
1. 第三方 API 暫時無法使用 → 切換到「財政部資料」
2. 財政部 API 需要權限 → 使用「第三方 API」或申請權限
3. 公司名稱或統編輸入錯誤 → 檢查輸入
4. 公司已歇業或註銷 → 資料庫可能沒有

### Q: 兩種 API 有什麼差別？

**A**: 
| 項目 | 第三方 API | 財政部資料 |
|------|-----------|-----------|
| 申請 | 不需要 | 需要申請 |
| 速度 | 快 | 稍慢 |
| 資料量 | 中等 | 完整 |
| 即時性 | 較高 | 視更新頻率 |

### Q: 如何申請經濟部 API？

**A**: 
1. 前往 https://data.gcis.nat.gov.tw/
2. 填寫使用告知書
3. 寄到 opendata.gcis@gmail.com
4. 等待 1-3 天審核

## 📱 功能說明

### 查詢方式
- 公司名稱（模糊搜尋）
- 統一編號（精確搜尋，8 位數字）

### 顯示資訊
- 統一編號

- 公司狀態
- 資本額
- 代表人
- 公司地址
- 設立日期
- 營業項目

## 🔧 進階設定

### 自訂 API Endpoint

如果你想使用其他 API，可以修改：
- `/app/api/search-thirdparty/route.ts`
- `/app/api/search-findata/route.ts`

### 新增環境變數

建立 `.env.local`:
```env
NEXT_PUBLIC_API_URL=your_api_url
```
